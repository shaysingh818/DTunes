# How to design software
This book explains it better than any dumbass professor/employer

1. Write the goal of the program in natural language (English)
2. Identify the (Nouns) of the problem statement / goals of the program
	* The nouns are the "data representation" of the program
3. Identify the "verbs" "action words" in the problem statement for the program
	* convert the verbs to "functions" or procedures

# SQLITE3 Object Relational Mapper
To avoid writting the same database methods tediously, We'll be creating a wrapper for sqlite3 in C. This is meant to act as a requirements document for what features/methods need to be implemented in order for the library to be fully modular. The sqlite3 wrapper will be used as a helper library for future projects. The next malware server I write will use this library for storing client connections.

# Delete

1. The delete method will generate a SQL query that deletes the model instance by id. If we want, we can add an optional unique identifier that we want the model to be generated by. For example, if we want to delete an audio file by "name", we should be able to do something like this, ```model.delete(name="shay")```.
